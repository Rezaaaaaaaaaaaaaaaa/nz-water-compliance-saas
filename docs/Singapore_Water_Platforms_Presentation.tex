\subsection{Example Integrated Workflow}

\textbf{Scenario: Contamination Event}

\begin{enumerate}[itemsep=0.15em]
    \item \textbf{Platform 1} detects water quality anomaly at sampling point
    \item Automatically alerts \textbf{Platform 2} to trace contamination source
    \item \textbf{Platform 2} simulates contaminant transport in network
    \item Identifies affected zones, generates isolation plan
    \item \textbf{Platform 3} checks if failing assets may be the source
    \item Operators execute valve closures through \textbf{Platform 2}
    \item \textbf{Platform 1} tracks remediation effectiveness
    \item \textbf{Platform 3} schedules asset replacement to prevent recurrence
\end{enumerate}

\textbf{Result:} 80\% faster response time, comprehensive root cause analysis, prevents future incidents
